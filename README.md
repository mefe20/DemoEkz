## Информационная система «Банкетам.Нет»

Проект разработан в рамках демонстрационного экзамена по компетенции «Разработчик веб и мультимедийных приложений» (Шифр варианта: В4 КОД 09.02.07-3-2026-ПУ).

## Описание системы
Проект представляет собой полноценное веб-приложение для автоматизации бронирования банкетных помещений (Зал, Ресторан, Летняя веранда, Закрытая веранда). Разработка выполнена с учетом современных стандартов фронтенд и бэкенд технологий на базе паттерна MVC и объектно-ориентированного программирования.

## Технологический стек
- **Backend:** Node.js, Express.js (REST API, маршрутизация, обработка сессий)
- **База данных:** PostgreSQL (реляционная структура, связи один-ко-многим)
- **Безопасность:** Библиотека `bcrypt` (хэширование конфиденциальных данных и паролей перед отправкой в СУБД)
- **Frontend:** HTML5, CSS3 (адаптивная сетка, селекторы, переменные среды), Ванильный JavaScript (асинхронное взаимодействие с API через Fetch, динамический рендеринг DOM-элементов)
- **Графическое оформление и UI/UX:** Интеграция локального шрифта `Oswald` согласно брендбуку, библиотеки иконок `Bootstrap Icons`, разработка интерактивного адаптивного слайд-шоу и кастомных модальных окон уведомлений (Toast)
- **Адаптивность:** Адаптивный дизайн, полностью оптимизированный под экраны мобильных устройств размером 390×844 пикселей с интеграцией интерактивного гамбургер-меню

## Инструкция по развертыванию приложения

1. **Установка зависимостей:**
   В корневой директории проекта выполнить команду:
   npm install
 
2. Настройка окружения:
Создать в корневой директории файл .env и указать параметры локального подключения к PostgreSQL:

Фрагмент кода
DB_USER=postgres
DB_PASSWORD=sa
DB_HOST=localhost
DB_PORT=5432
DB_NAME=banketam_db
PORT=3000

3. Инициализация базы данных:
Создать в pgAdmin 4 базу данных banketam_db и выполнить скрипт инициализации таблиц и связей, включая первичное заполнение справочника помещений.

4. Запуск приложения:
Для запуска в режиме разработки с автоматической перезагрузкой при изменениях файлов использовать команду:
    npm run dev
После этого веб-интерфейс будет доступен по локальному адресу: http://localhost:3000.
